Installation Instructions

  • Windows

    Install
    1. If DeadboltEdit is already installed, uninstall the older version first:
      • Go to Settings (Start->Settings) and select "Apps" or "Applications"
      • Locate "DeadboltEdit" in the list of installed apps, and select it
      • Choose "Uninstall"
        (This will be a standard Windows uninstall - just follow the prompts)
    2. Download the DeadboltEdit Installation program for Windows, and Save the file.
    3. Double-click the installer program, and follow the prompts.

    Uninstall
    1. Go to Settings (Start->Settings) and select "Apps" or "Applications"
    2. Locate "DeadboltEdit" in the list of installed apps, and select it
    3. Choose "Uninstall"
      (This will be a standard Windows uninstall - just follow the prompts)

  • Mac OS X

    Install
    1. If DeadboltEdit is already installed, you can install the new version in the same location - it will safely replace the older version.
      If you prefer, you can delete the older version first - see "Uninstall" below.
    2. Download the DeadboltEdit installation file for Mac OS X.
    3. Double-click the installation file (.dmg file). This should launch a new Finder window showing DeadboltEdit.app.
    4. Drag DeadboltEdit.app to your Applications folder.
    5. Eject (unmount) the .dmg file by clicking the eject icon in a Finder window.
      (you can then delete the .dmg file)
    6. Important! Beginning with Mac OS 10.8, a security feature called Gatekeeper only allows your Mac to run apps from the App Store, and apps that are signed by a certificate issued by Apple. (the certificate costs $100 USD per year) Gatekeeper may tell you this application is damaged - it isn't.
      To allow DeadboltEdit to run: Use Finder to view the Applications folder, then Ctrl+click or Right-click DeadboltEdit.app and choose "Open". It will run without warnings after doing this one time.
    7. Double-click DeadboltEdit.app to launch DeadboltEdit, or start it from Launchpad.

    Uninstall
    1. Delete DeadboltEdit.app in the Applications folder by moving it to Trash.

  • Linux - Installing from deb file on Ubuntu, Linux Mint, Debian:

    DeadboltEdit can be installed over an exixting version; There is no need to uninstall the old one.

    Download the DeadboltEdit deb file for 64-bit Linux.

    Install:
    • sudo dpkg -i {FULL_PATH_TO}/deadboltedit_3.50-1_amd64.deb

    Uninstall:
    • sudo dpkg -r deadboltedit

  • Linux - Installing from tar file:

    DeadboltEdit can be installed in any directory. Installing in /usr/share is recommended if you have admin rights to use the sudo command.

    Installing to the /usr/share directory will allow you to use the supplied launchers, which includes a "deadboltedit" command in /usr/bin, and a desktop launcher icon for Ubuntu-Unity, Gnome, or KDE desktop managers.

    Download the DeadboltEdit installation file for 64-bit Linux, and install as follows:

    Installing in /usr/share:
    1. If DeadboltEdit is already installed, uninstall the older version first. (See below)
    2. cd /usr/share
    3. sudo tar -xvpzf {FULL_PATH_TO}/DeadboltEdit_V3.50_Linux_64bit.tar.z
    4. sudo /usr/share/DeadboltEdit/install_launchers
    5. To run DeadboltEdit, use command /usr/bin/deadboltedit, or click on the application launcher icon (Unity, Gnome, or KDE desktops).

    Uninstall from /usr/share:
    1. sudo /usr/share/DeadboltEdit/remove_launchers
    2. sudo rm -rf /usr/share/DeadboltEdit


    Installing in directory other than /usr/share:
    1. If DeadboltEdit is already installed, uninstall the older version first. (See below)
    2. cd {INSTALLATION_DIRECTORY}
    3. tar -xvpzf {FULL_PATH_TO}/DeadboltEdit_V3.50_Linux_64bit.tar.z
    4. To run DeadboltEdit, use command {INSTALLATION_DIRECTORY}/deadboltedit

    Uninstall from directory other than /usr/share:
    1. rm -rf {INSTALLATION_DIRECTORY}/DeadboltEdit
